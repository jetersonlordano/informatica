Exercício 3: Planejamento de Férias

Objetivo: Criar uma planilha para planejar os custos de férias, utilizando soma e aplicação de descontos.

Instruções:

Siga os passos para organizar os custos de uma viagem de férias:

  1. Criação da Tabela: Insira os dados abaixo em uma nova planilha no Excel:
Item Categoria Quantidade Preço Unitário (R$) Valor Total (R$)
Passagens AéreasTransporte21.200,00
HotelHospedagem5 diárias300,00
AlimentaçãoComida5 dias100,00
PasseiosLazer3150,00
Transporte LocalTransporte5 dias50,00
  1. Calcular o Valor Total: Use a fórmula =C2*D2 para calcular o valor total de cada item e arraste para as demais linhas.
  2. Aplicar Desconto: Na célula ao lado da tabela, calcule o valor total da viagem (use =SOMA(E2:E6)) e aplique um desconto de 10% sobre o valor total.
  3. Formatação: Formate as células como "Moeda" e aplique formatação condicional para destacar valores acima de R$ 500,00.

Exercício 4: Controle de Notas Escolares

Objetivo: Criar uma planilha de controle de notas e calcular médias dos alunos.

Instruções:

Siga os passos para criar um controle de notas escolares no Excel:

  1. Criação da Tabela: Insira os dados abaixo em uma nova planilha:
Aluno Prova 1 Prova 2 Prova 3 Média Final
Ana8.59.07.5
Lucas7.06.58.0
Pedro9.58.09.0
Maria6.57.58.0
João8.07.07.5
  1. Calcular a Média: Use a fórmula =MÉDIA(B2:D2) para calcular a média final de cada aluno e arraste para os demais.
  2. Formatação Condicional: Aplique uma formatação condicional para destacar as médias acima de 7.0.
  3. Gráfico: Crie um gráfico de colunas para mostrar a média final de cada aluno.

Exercícios de Excel para Iniciantes

Exercício 1: Tabela de Despesas Mensais

Objetivo: Organizar dados de despesas em uma planilha e usar fórmulas simples para calcular soma e média.

Instruções:

Siga os passos abaixo para criar uma tabela de despesas mensais no Excel:

  1. Criação da Tabela: Insira os seguintes dados em uma nova planilha:
Dia Categoria Descrição Valor
01/09AlimentaçãoSupermercado120,00
03/09TransporteGasolina50,00
05/09LazerCinema30,00
10/09AlimentaçãoRestaurante60,00
12/09TransporteUber35,00
15/09EducaçãoCurso online200,00
20/09SaúdeFarmácia80,00
25/09LazerShow150,00
28/09EducaçãoLivros100,00
  1. Formatar a Planilha: Ajuste a largura das colunas, formate os valores como "Moeda", e destaque os títulos da tabela em negrito e cor de fundo.
  2. Inserir Fórmulas: Calcule o total das despesas usando a fórmula =SOMA(D2:D10) e a média diária com =MÉDIA(D2:D10).
  3. Gráfico: Crie um gráfico de barras para visualizar as despesas por categoria.

Exercício 2: Controle de Estoque

Objetivo: Criar uma planilha para controlar o estoque de produtos, aplicando fórmulas para calcular valores totais e médias.

Instruções:

Siga os passos para organizar o controle de estoque de uma loja:

  1. Criação da Tabela: Insira os dados abaixo em uma nova planilha no Excel:
Produto Categoria Quantidade Preço Unitário (R$) Valor Total (R$)
CamisetaVestuário5030,00
Calça JeansVestuário4090,00
NotebookEletrônicos102.500,00
SmartphoneEletrônicos151.200,00
TênisCalçados30150,00
  1. Calcular Valor Total: Insira a fórmula para calcular o valor total de cada produto: =C2*D2 e arraste para aplicar a todos os produtos.
  2. Calcular Estoque Total: Use a fórmula =SOMA(C2:C6) para calcular a quantidade total de produtos no estoque.
  3. Calcular Média de Preço: Utilize =MÉDIA(D2:D6) para calcular o preço médio dos produtos no estoque.
  4. Formatação: Ajuste as colunas e formate os valores como "Moeda". Deixe os títulos em negrito.
